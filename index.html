What I have learned

User Research

Is the start of every successful design. It helps add realistic context and insight to the user needs. My main focus when doing user research is to find the design opportunities that provide the best solutions for those needs.

User Interface Design

The main goal of my UI design is for it to be friendly and pleasurable for the user using key principles such as: hierarchy, accessibility, consistency, flexibility ad efficiency. Looking to create designs that are both aesthetic and minimalistic.

Web Development

My skills allow me to create websites with unique and creative approach to the users needs. Everything from technical back-end to the aesthetics need to create responsive web designs.
